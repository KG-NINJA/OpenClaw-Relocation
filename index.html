Workers Proxy

A security-first reverse proxy handling Cloudflare Access JWT validation and WebSocket interception.

Sandbox Container

Isolation at the edge. Running Node.js 22 with the Clawdbot Gateway in a firecracker-based VM.

R2 Persistence

Cron-driven 5-min sync ensures configuration and skills persist across container lifecycles.

// Persistent Storage Orchestration
async function syncToR2(sandbox, env) {
  const config = await sandbox.startProcess('cat /root/.clawdbot/config.json');
  await env.MOLTBOT_BUCKET.put('backup.json', config.stdout);
}

// Scheduled Cron Trigger
export default {
  scheduled: async (event, env) => {
    await syncToR2(sandbox, env);
  }
}

The Migration Challenges

OAuth Preservation

Existing OpenAI Codex sessions needed to be migrated without re-authentication in a browserless environment.

Solution: Profile Overlaying

Execution Tracing

Debugging "Code 1" crashes in a remote container requires deep visibility into shell initialization.

Solution: bash -x + 2>&1 proxy

WebSocket Shim

Proxying real-time chat data through Workers requires careful state management of WebSocket pairs.

Solution: Custom Proxy Handler
